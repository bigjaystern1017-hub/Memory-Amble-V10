Product direction:
MemoryAmble should feel less like a generic chatbot and more like a guided memory walk. The user is building a route inside a familiar place, planting vivid images, walking that route, and recalling what they placed. For now, we are only making the existing screens visually communicate that better.

Design target:
- Warm cream background
- White/near-white cards
- Soft borders
- Subtle shadows
- Large readable typography
- Purple primary CTA
- Simple line icons in soft circular pastel backgrounds
- Desktop web app layout, not mobile app layout
- Calm “Headspace for memory training” feel
- Timbuk should feel like a guide beside a lesson card, not a tiny chatbot avatar
